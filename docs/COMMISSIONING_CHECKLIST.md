# Commissioning checklist

## Dry checks
- [ ] Panel powers up
- [ ] PLC heartbeat visible
- [ ] HMI loads correct pages
- [ ] Every digital input forced and verified
- [ ] Every analog input checked for sane scaling
- [ ] Every valve commanded and verified
- [ ] Every pump interlock tested
- [ ] Alarm beacon and buzzer tested
- [ ] Backup-power switchover tested

## Clean-water checks
- [ ] Leak check complete
- [ ] Flow meter agrees with timed-volume spot check
- [ ] Separator and waste path stay dry externally
- [ ] Cartridge housings vent and drain correctly
- [ ] Final discharge and recirculation routes both verified
- [ ] Stale-data timeout forces blocked discharge

## Contaminated-water trial checks
- [ ] Sample IDs prepared
- [ ] PPE and waste path ready
- [ ] Influent / interstage / effluent samples collected
- [ ] Visual free-oil observations logged
- [ ] Operator notes retained
- [ ] Final event log exported
