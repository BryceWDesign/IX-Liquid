# Sources used for budgetary planning

This repo uses representative market examples to ground budgetary planning for a pilot build. Pricing is volatile and should be re-checked before procurement.

Representative references consulted for planning:
- small skid-mounted water-treatment build and startup context
- low-cost PLC hardware pages for CLICK / CLICK PLUS
- representative UV system pricing near the 20-21 GPM class
- representative industrial water-quality instrumentation pricing for pH / ORP / conductivity kits
- representative Big Blue housing pricing

These were used only to set planning ranges, not as procurement endorsements.
