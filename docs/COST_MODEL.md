# Cost model

## Summary

The budgetary BOM in this repo totals:

- **Low estimate:** $40,372
- **High estimate:** $66,515

This is not a vendor quote. It is a **planning-level pilot-build estimate** for an original skid based on representative off-the-shelf hardware classes, fabrication labor, controls work, and commissioning.

## Cost posture

IX-Liquid is intentionally scoped as a **small pilot skid**. A field-capable pilot is much cheaper than a permanent industrial plant, but more expensive than a hobby prototype because it must include:
- real separation hardware
- real valves and containment
- serviceable layout
- instrumentation
- controls hardware
- panel build
- assembly labor
- wet testing
- sample/lab budget
- contingency

## Main cost drivers

1. Coalescing separator package
2. Fabrication labor
3. Controls programming and HMI
4. Polishing media train
5. Instrumentation
6. Commissioning and sample plan

## Build classes

### Minimal bench demonstrator
- Target range: **$15k-$25k**
- Suitable for logic and routing proof only
- Not recommended as the repo's flagship build

### Serious field pilot
- Target range: **$40k-$67k**
- Matches this repo's BOM and assembly instructions
- Best balance of realism and bounded scope

### Trailerized polished unit
- Target range: **$55k-$90k**
- Better packaging, weather hardening, nicer operator interface, more deployment polish
